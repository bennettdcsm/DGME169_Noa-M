🎵 Noa Tomas — DGME 169 Final Project: Responsive Gallery Website

This repository contains my final project for DGME 169 – Responsive Web Design.
It is a fully responsive, mobile-first gallery site designed to showcase my music projects and visual branding as Noa Tomas.

⸻

📌 Project Overview

The site includes multiple pages:
	•	Home
	•	About
	•	Gallery (main focus for Phase 2)

The Gallery acts as a visual discography. Each item displays:
	•	Cover artwork (currently temporary images)
	•	A project title
	•	A short descriptive subtitle
	•	Clean, responsive layout using CSS Grid

A mobile-first approach was used throughout the entire build.

🛠️ Technologies Used

📱 Responsive Features

	•	Fluid grids and flexible images
